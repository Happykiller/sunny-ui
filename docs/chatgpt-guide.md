# AI Usage Guide

This guide helps AI assistants answer questions about Sunny UI.

- Always reference the repository files when providing code snippets or explanations.
- Use the [User Guide](./user-guide.md) and [Contributor Guide](./contributor-guide.md) for contextual information.
- Component reference files are stored under [docs/components](./components).
- The implementation lives in the `src` directory and builds with `npm run build`.
- When quoting source lines, include the path and relevant line numbers.
